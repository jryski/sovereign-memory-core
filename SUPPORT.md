# Support

Sovereign Memory Core is early alpha/reference implementation work.

Use GitHub issues for:

- bugs
- documentation fixes
- validation gaps
- conformance gaps
- small implementation tasks
- research tasks

GitHub Discussions may be used later if external design discussion becomes useful.

There is no production support guarantee. Do not use this repository with sensitive production data without reviewing the security model and validating the operational path in a local/disposable environment first.
